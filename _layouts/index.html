Introducing
RSTO Game Token

RSTO is an application token that allows users access to the Roostoo product. The main utility will be for users to enter mock crypto trading competitions hosted on the Roostoo platform. Users will be able to receive RSTO back according to their performances. It will be ideal to develop RSTO as a utility token for using the Roostoo application as well as an on-chain cash flow token.
